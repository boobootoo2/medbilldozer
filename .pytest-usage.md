# Quick Pytest Usage Guide

## Running Tests

Since pytest is installed via pip but not in your shell PATH, use `python3 -m pytest`:

### Run All Tests
```bash
python3 -m pytest tests/ -v
```

### Run Specific Test File
```bash
# Configuration tests
python3 -m pytest tests/test_config.py -v

# Documentation Assistant tests
python3 -m pytest tests/test_doc_assistant.py -v
```

### Run Specific Test Class
```bash
python3 -m pytest tests/test_config.py::TestAppConfig -v
python3 -m pytest tests/test_doc_assistant.py::TestGetAnswerOpenAI -v
```

### Run Specific Test Function
```bash
python3 -m pytest tests/test_config.py::TestAppConfig::test_default_config_structure -v
```

### Run with Different Output Options
```bash
# Quiet mode (less output)
python3 -m pytest tests/ -q

# Show print statements
python3 -m pytest tests/ -s

# Show only summary
python3 -m pytest tests/ --tb=no

# Show detailed failures
python3 -m pytest tests/ --tb=long
```

### Run Tests by Marker
```bash
# Run only unit tests
python3 -m pytest tests/ -m unit

# Run only integration tests
python3 -m pytest tests/ -m integration
```

### Collect Tests Without Running
```bash
python3 -m pytest tests/ --collect-only
```
